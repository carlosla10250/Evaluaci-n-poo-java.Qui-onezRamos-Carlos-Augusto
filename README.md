# Sistema de gestion de tareas escolares

Proyecto realizado en Java usando Programacion Orientada a Objetos.

## Funciones
1. Crear tareas
2. Ver tareas
3. Marcar tareas entregadas
4. Ver porcentaje de avance

## Requisitos
- JDK 21
- IntelliJ IDEA

## Como ejecutar
1. Abrir el proyecto
2. Ejecutar Main.java
